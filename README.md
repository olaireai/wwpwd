# What Would Paul Weller Do?

A stylish question and answer app that generates fictional advice in the spirit of mod culture, sharp style, and British music heritage.

Built with Next.js, TypeScript, Tailwind CSS, and the Anthropic API.

---

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Add your API key

Copy the example env file and add your Anthropic API key:

```bash
cp .env.example .env
```

Then edit `.env` and replace `your-api-key-here` with your actual key from [console.anthropic.com](https://console.anthropic.com/).

### 3. Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Railway

1. Go to [railway.app](https://railway.app) and create a new project.
2. Choose **Deploy from GitHub repo** and select this repository.
3. In the Railway project settings, add an environment variable:
   - `ANTHROPIC_API_KEY` = your Anthropic API key
4. Railway will auto-detect the Next.js project and deploy it.
5. Once deployed, Railway provides a public URL.

No Dockerfile or Procfile needed — Railway handles Next.js natively.
